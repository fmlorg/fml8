関数名および変数名に関連した用語のまとめ
用語/単語の省略形 説明
fh ファイルハンドル( file handle )
dh ディレクトリハンドル( directory handle )
fp 関数へのポインタ( function pointer )
ただし perl では CODE リファレンスと呼ぶべき
_なになに 慣習的にパッケージ、クラス、オブジェクト内部で使う 変数や関数名に _ ではじまるキーワードを使う。
curproc ”current process 構造体”の略
rなになに r はリファレンスの略。ハンガリアン記法
特別に強調したい場合に使うことがあるが、 あまり使っていない。
STR 文字列 ( string ) の略
日本語コードの略称 JIS SJIS EUC などおなじみのもの…
2 to と発音が一緒のため省略形としてよく使う。 類似品として 4 ( for の意味)もある
例: STR2EUC (STRing to EUC の略) 
		use String qw(STR2EUC);
		$euc_string = &STR2EUC($string);
4 for と発音が同じために用いる。
